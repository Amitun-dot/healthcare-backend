# 🏥 Healthcare Appointment & Prescription System (Backend)

A Spring Boot backend for managing healthcare operations including appointment booking, prescription generation, and user management with secure JWT authentication.

---

## 🚀 Live Demo

- ⚙️ Backend: https://healthcare-backend-5mtt.onrender.com

---

## ✨ Features

- 🔐 JWT-based authentication & authorization
- 👥 Role-based access (ADMIN, DOCTOR, PATIENT)
- 📅 Appointment booking & management
- 🚫 Prevents duplicate appointment slots
- 📄 Prescription generation (PDF)
- 📧 Email integration for sending prescriptions
- 🛠️ Admin dashboard APIs

---

## 🛠️ Tech Stack

- Spring Boot
- Spring Security
- Spring Data JPA (Hibernate)
- MySQL
- JWT Authentication

---
